      SingleLineTest Class

      Contains unit tests for verifying the correct behavior of single-line commands in the Boose application. This class includes tests for drawing and movement commands such as:
      • moveto
      • drawto
      • circle
      • rect
      • tri
      • clear
      • reset
      Each test ensures that the canvas behaves as expected after running the command, checking pen positions and behavior.
